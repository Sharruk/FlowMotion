2️⃣ Habit Reminder Widget

This widget reminds the user to perform a habit today.

Instead of showing days, it shows status.

Example:

Drink Water

✔ Completed Today

or

Workout

⚠ Not Completed Today

or

Meditation

Next reminder: 6 PM

This widget connects directly to the habit system in your app.

Example habits in **FlowMotion:

Drink Water
Workout
Read 30 Minutes
Meditation

If user completed the habit → widget shows:

✔ Done

If not completed → widget shows:

⚠ Pending

So the widget answers:

“Did I complete this habit today?”

This is very useful in productivity apps.

Real Floating Desktop Widget

Instead of opening Mozilla Firefox, you can create a floating desktop widget window.

Example:

On your **Ubuntu desktop:

-------------------
 Hackathon

   ○ progress ring

   14
   Days
-------------------

This widget:

• stays on the desktop
• updates automatically
• does NOT open browser
• looks like a real desktop gadget

Similar to:

• Windows desktop gadgets
• Android widgets
• macOS widgets

How This Works Technically

Instead of .desktop shortcut → browser,

you run a Python widget app.

Example technologies:

• Python
• Tkinter or GTK
• Reads data from Django database

Example architecture:

FlowMotion (Django server)
        │
        │ database
        ▼
Desktop Widget Script
        │
        ▼
Floating Window Widget

So the widget window reads the countdown from the database and updates it.

Example Widget Window
┌─────────────────┐
│   Hackathon     │
│                 │
│     ○ progress  │
│                 │
│      14         │
│      Days       │
